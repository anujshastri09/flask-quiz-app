# Flask Quiz App

A simple Flask Quiz application with login/signup, category selection, multiple choice questions, and score tracking.

## How to Run

1. Create a virtual environment:
python -m venv venv

2. Activate virtual environment:


- Windows: `venv\Scripts\activate`
- Linux/Mac: `source venv/bin/activate`

3. Install dependencies:
pip install -r requirements.txt


4. Run the app:
python app.py
